##title换行设置       我们在设置导航栏标题时，通常只需要设置navigationItem的title属性即可。但是，该属性只能设置单行的标题，若想要设置多行，则需要通过自定义view（即navigationItem.titleView）来实现。
下面我们来举例说明如何实现多行标题:
第一步，自定义UILabel ~~~~{ruby} UILabel *titleView = [[UILabel alloc] initWithFrame:CGRectMake(0, 0, 200, 44)]; titleView.textAlignment = NSTextAlignmentCenter; // 注： titleView.numberOfLines = 0; NSString *titleStr = @"发布微博\n hanmeng"; ~~~~ 第二步，定义一个可变的NSMutableAttributedString，便于添加 ~~~~{ruby} NSMutableAttributedString *attributes = [[NSMutableAttributedString alloc] initWithString:titleStr]; ~~~~ 第三步，添加属性 ~~~~{ruby} [attributes addAttribute:NSFontAttributeName value:[UIFont systemFontOfSize:16] range:NSMakeRange(0, 4)]; [attributes addAttribute:NSFontAttributeName value:[UIFont systemFontOfSize:12] range:NSMakeRange(6, titleStr.length - 6)]; [attributes addAttribute:NSForegroundColorAttributeName value:[UIColor orangeColor] range:NSMakeRange(6, titleStr.length - 6)]; ~~~~ 第四步，赋值 ~~~~{ruby} titleView.attributedText = attributes; self.navigationItem.titleView = titleView; ~~~~ 最终效果图如下：

    [attributes addAttribute:NSForegroundColorAttributeName value:[UIColor orangeColor] range:NSMakeRange(6, titleStr.length - 6)];
