Swift2.2中警告提示

      OC中可以通过#warning方法来提醒自己将来需要注意的地方，但是在使用Swift创建项目时，目前Xcode版本（本人使用的Xcode 7.3版本）没有这样的功能产生黄色警告。

      这时候就需要我们自己手动去让Xcode产生这个功能了。接下来就来介绍下如何开启这个功能：
第一步：在项目内部找到Build Phases，选择坐标的“+”号，如下图所示：upload image now! 第二步：添加 New Run Script Phase，点击了之后就出现了下图的Run Script栏upload image now! 第三步：在蓝色区域添加下面代码

TAGS="TODO:|FIXME:|WARNING:"
echo "searching ${SRCROOT} for ${TAGS}"
find "${SRCROOT}" ( -name ".swift" ) -print0 | xargs -0 egrep --with-filename --line-number --only-matching "($TAGS).\$" | perl -p -e "s/($TAGS)/ warning: \$1/"
upload image now! 根据个人不同的需求，TODO: 和 FIXME: 是可以在代码中改变的。 记住TODO和分号之后不能有空格哦 最终效果图如下：upload image now!
