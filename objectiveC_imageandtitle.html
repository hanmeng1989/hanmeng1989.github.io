##图文混排 显示文字：UILabel/UIButton
显示图片：UIImageView和UIButton
显示图文混排：UILabel、UITextView、UITextField
      图文混排技术的实现，主要通过修改控件的`attributedText`属性来实现。本文以UILabel为例介绍如何利用UILabel来展示图文混排。

图文混排

显示文字：UILabel/UIButton
显示图片：UIImageView和UIButton
显示图文混排：UILabel、UITextView、UITextField

      图文混排技术的实现，主要通过修改控件的attributedText属性来实现。本文以UILabel为例介绍如何利用UILabel来展示图文混排。
最终效果图：

第一步：创建Label并添加到View

// 1.创建Label
UILabel *label = [[UILabel alloc] initWithFrame:CGRectMake(100, 100, 100, 100)];

[self.view addSubview:label];

第二步： 图文混排
2.1 初始化attribute,并添加文字 “I ”

 // 2.1 初始化attribute,并添加文字 “I ”
 NSMutableAttributedString *mutableAttribute = [[NSMutableAttributedString alloc] init];

 [mutableAttribute setAttributedString:[[NSAttributedString alloc] initWithString:@"I "]];

2.2 添加图片

 // 2.2 添加图片
 NSTextAttachment *attachment = [[NSTextAttachment alloc] init];

 attachment.image = [UIImage imageNamed:@"d_aini"];

 NSAttributedString *attri = [NSAttributedString attributedStringWithAttachment:attachment];

 [mutableAttribute appendAttributedString:attri];

2.3 添加另外一部分文字 @" You"

// 2.3 添加另外一部分文字 @" You"
 [mutableAttribute appendAttributedString:[[NSAttributedString alloc] initWithString:@" You"]];

第三步：赋值给label的attributedText

label.attributedText = mutableAttribute;

最近在学Swift，所以用Swift语言又实现了一下，代码如下：

// 1.创建lbl并添加到View
 let lbl = UILabel(frame: CGRectMake(100, 100, 100, 100))

 view.addSubview(lbl)

 // 2.图文混排

 // 2.1 文字
 let mulAttribute = NSMutableAttributedString()

 mulAttribute.setAttributedString(NSAttributedString(string:"I "))

 // 2.2 图片
 let attachment = NSTextAttachment()

 attachment.image = UIImage(named: "d_aini")

 let attribute = NSAttributedString(attachment: attachment)

 mulAttribute.appendAttributedString(attribute)

 // 2.3 文字
 mulAttribute.appendAttributedString(NSAttributedString(string: " You"))

 // 3.赋值给label的attributedText
 lbl.attributedText = mulAttribute
